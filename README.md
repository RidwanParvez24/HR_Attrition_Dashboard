# 👥 HR Analytics Dashboard - Employee Attrition

Comprehensive Human Resources analytics dashboard for tracking employee attrition, workforce metrics, and retention analysis to optimize talent management strategy.

## 🎯 Project Overview

This HR analytics project provides detailed insights into employee attrition patterns, retention metrics, and workforce analytics to help organizations make data-driven HR decisions and improve employee satisfaction.

## 📈 Key Features

- **Attrition Analysis**: Monitor employee turnover rates and trends
- **Department Insights**: Analyze attrition by department and role
- **Retention Metrics**: Track employee retention and satisfaction
- **Demographic Analysis**: Understand attrition patterns by demographics
- **Predictive Insights**: Identify at-risk employee segments
- **Workforce Planning**: Support strategic HR decisions

## 🛠️ Technologies Used

- **BI Platform**: Power BI, Tableau, or Excel
- **Data Processing**: Python, SQL
- **Database**: SQL Server, MySQL
- **Visualization**: Power BI Desktop, Tableau
- **Tools**: Jupyter Notebook, Excel

## 📊 Dashboard Components

- **Executive Summary**: Key HR metrics overview
- **Attrition Rate**: Overall and trend analysis
- **Departmental Breakdown**: Attrition by department
- **Tenure Analysis**: Employee tenure distribution
- **Compensation Analysis**: Salary vs attrition correlation
- **Performance Metrics**: Performance rating analysis
- **Risk Assessment**: Identifying flight risks

## 🚀 Getting Started

1. **Load Employee Data**
   ```bash
   Place raw HR data in data/employee_data/ folder
   ```

2. **Prepare Data**
   ```bash
   python scripts/data_preparation.py
   ```

3. **Open Dashboards**
   - HR_dashboard.pbix - Main dashboard
   - Attrition_report.pbix - Detailed analysis

4. **Analyze Insights**
   - Review trends and patterns
   - Identify retention opportunities

## 💡 Key Metrics

| Metric | Description |
|--------|-------------|
| Attrition Rate | Percentage of employees leaving |
| Retention Rate | Percentage of employees staying |
| Average Tenure | Mean years of service |
| Turnover Cost | Financial impact of attrition |
| Flight Risk Index | Probability of employee departure |
